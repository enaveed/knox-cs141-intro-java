
# Fruit Catcher Game (Java, Greenfoot)

This project is a simple arcade-style **Fruit Catcher game** developed using **Java** and the **Greenfoot** framework.  
The player controls a bucket to catch falling fruits while avoiding missed catches that end the game.

- Fruits fall from the top of the screen at random positions.
- The player moves a bucket left and right to catch the fruits.
- Each successful catch increases the score.
- Missing a fruit results in a game-over condition.
